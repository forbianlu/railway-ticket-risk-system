# 后续开发路线

## 第 1 周：能跑起来

- 安装 Maven
- 启动 Spring Boot 后端
- 打开前端页面完成一次查车、购票、退票
- 截图保存到 README

## 第 2 周：补基础能力

- 学会解释 Controller、Service、Repository 的职责
- 学会看 H2 控制台里的表数据
- 给订单创建和退票补单元测试
- 将数据库从 H2 切换到 MySQL

## 第 3 周：简历增强

- 增加 Redis 热门车次缓存
- 将演示版用户登录和角色权限升级为 Spring Security + JWT + BCrypt
- 增加接口文档和数据库 ER 图
- 配置 GitHub Actions 自动测试

## 第 4 周：面试准备

- 整理项目亮点和难点
- 准备 10 个高频问题答案
- 练习手写订单表、库存表和风险事件表设计
- 练习讲清楚一次下单的完整链路
